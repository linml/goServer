config.json 配置项说明：	

	CurIndex   当前轮到的玩家
	SiteIndex  座位
	CurCard    当前出的牌
	Feng       圈风
	List       手牌
	Chow       吃牌
	Peng       碰牌
	Kong       杠牌
	KongType   杠牌类型 0=明杠 1=暗杠 2=碰杠
	Hua        花牌
	Hu         胡牌
	Ting       是否听牌
	ChengBao   承包
	StartIndex int   //开始的牌的位置
	CurMJIndex int   //当前拿牌的位置 (断线重连)
	EndBlank   int   //结尾拿掉的牌 (杠后从结尾拿掉的牌)
